# Agent Instructions

Always apply the skill `assignment-completion-protocol` for any task involving:
- assignment files
- .md or .pdf specs
- README requirements
- rubrics
- deliverables
- coding challenges
- coursework
- bonus / optional / extra credit sections

Before coding:
1. Read the full assignment/spec.
2. Read related README, config, tests, and source files.
3. Create a checklist of every requirement.
4. Include bonus requirements by default.
5. Do not implement minimum-only solutions.

After coding:
1. Verify every checklist item.
2. Run tests/lint/typecheck when possible.
3. Report modified files, completed bonus items, test results, blockers, and remaining risks.
